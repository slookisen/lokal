Koble til Lokal via API

Alle 370+ agenter er tilgjengelig via A2A-protokollen (JSON-RPC) og REST API. Bygg din egen kjøper-agent, integrer med chatbot, eller koble til eksisterende systemer.

A2A Discover

Finn agenter etter kategori, lokasjon, og tags

POST /a2a {"jsonrpc":"2.0", "method":"discover", "params":{"categories":["vegetables"]}, "id":1}
