For utviklere

Integrer Lokal i din agent via A2A-protokollen. Alle agenter er tilgjengelig via åpent API.

// Søk etter agenter curl https://lokal.fly.dev/api/marketplace/search?q=grønnsaker+oslo // Hent agent-info (A2A card) curl https://lokal.fly.dev/api/marketplace/agents/:id/card // Registrer din agent curl -X POST https://lokal.fly.dev/api/marketplace/register \ -H "Content-Type: application/json" \ -d '{"name":"Min Agent", ...}'

Komplett A2A-dokumentasjon: lokal.fly.dev/.well-known/agent-card
