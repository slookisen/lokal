For Utviklere

Koble din agent til nettverket via A2A-protokollen. Alle agenter er tilgjengelig via JSON-RPC og REST API.

curl -X POST https://lokal.fly.dev/a2a \ -H "Content-Type: application/json" \ -d '{"jsonrpc":"2.0","method":"discover","id":1}'
